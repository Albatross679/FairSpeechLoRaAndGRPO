**1. `estimate_adapter_vram_mb(rank, n_targets, hidden_dim=1536, n_layers=28)`**
Theoretical VRAM per adapter. Formula:
- params_per_layer = 2 * hidden_dim * rank * n_targets (A + B matrices)
- total_params = params_per_layer * n_layers
- bytes_per_param = 2 (bf16) + 8 (AdamW: 2 fp32 states) + 2 (gradients bf16) = 12
- activation_overhead_mb = 300 (assumed, will be calibrated by profiling)
- vram_mb = (total_params * bytes_per_param) / (1024**2) + activation_overhead_mb
- Return vram_mb as float
- Print breakdown: param count, weight memory, optimizer memory, gradient memory, activation estimate

**2. `compute_pack_size(configs, available_vram_mb=5000, safety_margin_mb=0)`**
How many adapters from `configs` list fit in `available_vram_mb`.
- `configs` is a list of dicts with keys: rank, target_mlp (bool)
- For each config: n_targets = 4 (attn only) or 7 (attn+mlp) based on target_mlp
- Greedily add configs by calling `estimate_adapter_vram_mb()` until budget exceeded
- Always return at least 1 (fallback to sequential)
- Return (pack_size: int, estimated_total_mb: float, per_adapter_mb: list[float])

**3. `profile_adapter_vram(model=None, rank=4, target_mlp=False, n_steps=3)`**

**Method: setup_adapters()**
- The first adapter in adapter_configs is assumed already applied via get_peft_model() during model loading. Its LoRA config is already on the model.
- For adapters [1:]: call `model.add_adapter(name, LoraConfig(r=rank, lora_alpha=alpha, lora_dropout=dropout, target_modules=targets, task_type=TaskType.CAUSAL_LM))` where targets = ["q_proj","k_proj","v_proj","o_proj"] + (["gate_proj","up_proj","down_proj"] if target_mlp)
- After adding all adapters, re-freeze audio_tower: `for name, param in model.named_parameters(): if "audio_tower" in name: param.requires_grad = False`
- Create per-adapter optimizers: for each adapter, `model.set_adapter(name)`, collect trainable params `[p for n,p in model.named_parameters() if p.requires_grad and "lora" in n.lower()]`, create `torch.optim.AdamW(params, lr=config["lr"], weight_decay=config["weight_decay"])`
- Create per-adapter DataLoaders: for each adapter, create an `ASRFairnessDataset` from train_df (save train_df to a temp CSV, pass to ASRFairnessDataset), with `DataCollatorForQwen3ASR(processor)` as collate_fn, batch_size=per_device_batch_size, shuffle=True
- Create per-adapter data iterators (infinite cycling): wrap each DataLoader in an `iter()`, use a helper that calls `next()` and re-creates iter on StopIteration
- Create eval DataLoader similarly from eval_df (shared across adapters)
- Store: self.active_adapters (list of names), self.optimizers (dict), self.data_iters (dict), self.losses (dict of lists), self.eval_losses (dict of lists of (step, loss) tuples)

**Method: train_step(step)**
Round-robin through all active adapters:
```python
for name in self.active_adapters:
    self.model.set_adapter(name)
    self.model.train()
    batch = next(self.data_iters[name])
    # Move batch to GPU
    batch = {k: v.to(self.model.device) if isinstance(v, torch.Tensor) else v
             for k, v in batch.items()}
    self.optimizers[name].zero_grad()
    with torch.autocast("cuda", dtype=torch.bfloat16):
        outputs = self.model(**batch)
        loss = outputs.loss
    loss.backward()
    self.optimizers[name].step()
    self.losses[name].append(loss.item())
```

**Method: evaluate(step)**
Evaluate all active adapters:
```python
for name in self.active_adapters:
    self.model.set_adapter(name)
    self.model.eval()
    total_loss = 0.0
    n_batches = 0
    with torch.no_grad():
        for batch in self.eval_loader:
            batch = {k: v.to(self.model.device) if isinstance(v, torch.Tensor) else v
                     for k, v in batch.items()}
            with torch.autocast("cuda", dtype=torch.bfloat16):
                outputs = self.model(**batch)
            total_loss += outputs.loss.item()
            n_batches += 1
    avg_loss = total_loss / max(n_batches, 1)
    self.eval_losses[name].append((step, avg_loss))
    print(f"    [{name}] step {step}: eval_loss={avg_loss:.4f}")
```
Print VRAM after evaluation via `torch.cuda.max_memory_allocated()`.

**Method: remove_adapter(name)**
- Remove from active_adapters list
- Delete optimizer: `del self.optimizers[name]`
- Delete data iterator: `del self.data_iters[name]`
- Call `model.delete_adapter(name)` if PEFT supports it, otherwise just deactivate

**Method: train(n_steps, eval_steps, rung_steps=None)**
Main training loop:
```python
for step in range(1, n_steps + 1):
    self.train_step(step)
    if step % eval_steps == 0 or step == n_steps:
        self.evaluate(step)
        # Print VRAM
        if torch.cuda.is_available():
            vram_mb = torch.cuda.max_memory_allocated() / (1024**2)
            print(f"  Step {step}: peak VRAM = {vram_mb:.0f} MB")
    if rung_steps and step in rung_steps:
        # Yield rung results for pruning decisions
        yield step, {name: self.eval_losses[name][-1][1]
                     for name in self.active_adapters
                     if self.eval_losses[name]}
```

Main sweep loop using Optuna ask/tell API:

```python
def run_packed_sweep(args):
    stage = args.stage
    n_trials = args.n_trials
    steps = args.steps_per_trial
    eval_steps = DEFAULT_EVAL_STEPS

    # 1. Create subset (same as Phase 2: 500 samples, 10% eval)
    print(f"Creating stratified subset (size={args.subset_size})...")
    subset_df = create_stratified_subset(
        args.fs_manifest, args.cv_manifest, args.subset_size, seed=SEED)
    n_eval = max(10, int(len(subset_df) * DEFAULT_EVAL_FRACTION))
    eval_df = subset_df.tail(n_eval).reset_index(drop=True)
    train_df = subset_df.head(len(subset_df) - n_eval).reset_index(drop=True)

    os.makedirs(args.output_dir, exist_ok=True)
    train_df.to_csv(os.path.join(args.output_dir, "sweep_train.csv"), index=False)
    eval_df.to_csv(os.path.join(args.output_dir, "sweep_eval.csv"), index=False)
    print(f"  Train: {len(train_df)}, Eval: {len(eval_df)}")

    # 2. Create Optuna study
    pruner = optuna.pruners.MedianPruner(n_startup_trials=5)  # Stage 1
    study = optuna.create_study(
        direction="minimize",
        sampler=optuna.samplers.TPESampler(seed=SEED),
        pruner=pruner,
        study_name=f"packed-sweep-stage{stage}",
    )

    # 3. Load processor once
    _, processor = load_model_and_processor()

    # 4. Ask/tell loop with packing
    all_results = []
    completed = 0
    t0 = time.time()

    while completed < n_trials:
        remaining = n_trials - completed
        
        # Ask batch of trials
        pending_trials = []
        pending_configs = []
        for _ in range(min(remaining, 8)):  # ask up to 8 at a time
            trial = study.ask()
            params = suggest_hps(trial, stage)
            pending_trials.append(trial)
            pending_configs.append({
                "name": f"trial_{trial.number:03d}",
                **params,
            })

        # Compute pack size based on configs
        pack_size, est_total, per_adapter = compute_pack_size(
            pending_configs, available_vram_mb=AVAILABLE_FOR_ADAPTERS_MB)
        pack_size = min(pack_size, len(pending_configs))

        # Take first pack_size configs for this batch
        batch_trials = pending_trials[:pack_size]
        batch_configs = pending_configs[:pack_size]

        # Return unused trials to Optuna (re-ask later)
        # NOTE: Optuna ask/tell doesn't have a "return" -- we'll just tell
        # the unused ones as pruned and re-ask. Simpler: just ask exactly
        # pack_size trials at a time. Revise the loop:
        # Actually, simpler approach: ask one batch of pack_size trials.
        
        print(f"\n{'='*60}")
        print(f"Pack batch: {len(batch_configs)} adapters "
              f"(est {est_total:.0f} MB, budget {AVAILABLE_FOR_ADAPTERS_MB} MB)")
        for c in batch_configs:
            print(f"  {c['name']}: rank={c['rank']}, mlp={c['target_mlp']}, "
                  f"lr={c['lr']:.2e}")
        print(f"{'='*60}")

        # Load fresh model + apply first adapter
        model, _ = load_model_and_processor()
        model = create_packed_model(model, batch_configs[0])
        if torch.cuda.is_available():
            model = model.cuda()

        # Rename default adapter to first config's name
        # PEFT default adapter name after get_peft_model is "default"
        # We'll keep "default" as the first adapter name internally,
        # but map it in results
        batch_configs[0]["name"] = "default"

        # Create PackedTrainer
        trainer = PackedTrainer(
            model=model,
            adapter_configs=batch_configs,
            processor=processor,
            train_df=train_df,
            eval_df=eval_df,
            steps_per_trial=steps,
            eval_steps=eval_steps,
            per_device_batch_size=2,
            seed=SEED,
        )
        trainer.setup_adapters()

        # Train
        for step, rung_results in trainer.train(
            n_steps=steps, eval_steps=eval_steps):
            pass  # Stage 1: no pruning mid-training

        # Collect results
        results = trainer.get_results()
        for trial, config in zip(batch_trials, batch_configs):
            name = config["name"]
            if name in results:
                eval_loss = results[name]
                study.tell(trial, eval_loss)
                all_results.append({
                    "trial_number": trial.number,
                    "eval_loss": eval_loss,
                    "params": {k: config[k] for k in
                               ["rank", "alpha_ratio", "alpha", "dropout",
                                "target_mlp", "lr", "weight_decay"]},
                })
                print(f"  {name}: eval_loss={eval_loss:.4f}")
            else:
                study.tell(trial, float("inf"),
                           state=optuna.trial.TrialState.FAIL)

        completed += len(batch_trials)

        # Cleanup
        del model, trainer
        gc.collect()
        if torch.cuda.is_available():
            torch.cuda.empty_cache()

    elapsed = time.time() - t0

    # 5. Save results
    save_results(study, all_results, elapsed, args)

    # 6. Gate validation (Stage 1 only)
    if stage == 1 and args.phase2_results:
        validate_against_phase2(study, args)

    return study
```
