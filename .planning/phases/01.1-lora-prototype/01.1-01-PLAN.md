Create `scripts/training/data_collator.py` implementing `DataCollatorForQwen3ASR`:

```python
class DataCollatorForQwen3ASR:
    def __init__(self, processor):
        self.processor = processor

    def __call__(self, features):
        # features is a list of dicts from ASRFairnessDataset:
        #   {"audio": tensor(samples,), "transcript": str, "group": str}
        #
        # 1. For each feature, build the chat message in Qwen3-ASR format:
        #    messages = [
        #        {"role": "user", "content": [{"type": "audio", "audio": audio_numpy}]},
        #        {"role": "assistant", "content": "language English<asr_text>" + transcript},
        #    ]
        # 2. Apply processor with add_generation_prompt=False to get input_ids, attention_mask, input_features, feature_attention_mask
        # 3. Create labels by copying input_ids, then mask prefix tokens with -100
        #    (prefix = everything before the assistant's transcript content)
        #    Find the position of "language English<asr_text>" token sequence,
        #    set labels[:, :prefix_end] = -100
        # 4. Pad all sequences in the batch to the same length
        # 5. Return dict with keys: input_ids, attention_mask, input_features,
        #    feature_attention_mask, labels
```

The prefix masking is critical (per RESEARCH.md Pitfall 6) — loss must only be computed on transcript tokens, not on the system/user prompt tokens. Use the processor's tokenizer to find where the assistant response starts.

Forward patch (per D-15, RESEARCH.md Pattern 2):
```python
def patch_outer_forward(model):
    cls = model.__class__
    if getattr(cls, "_forward_patched", False):
        return
    if not hasattr(model, "thinker") or not hasattr(model.thinker, "forward"):
        raise RuntimeError("Cannot patch forward: model has no .thinker.forward")
    def forward(self, input_ids=None, attention_mask=None,
                input_features=None, feature_attention_mask=None,
                labels=None, **kwargs):
        return self.thinker.forward(
            input_ids=input_ids, attention_mask=attention_mask,
            input_features=input_features,
            feature_attention_mask=feature_attention_mask,
            labels=labels, **kwargs,
        )
    cls.forward = forward
    cls._forward_patched = True
```
