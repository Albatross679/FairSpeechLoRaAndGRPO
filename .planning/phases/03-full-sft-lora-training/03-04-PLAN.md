**Step 1: Split eval_subset.csv by dataset source.**
The eval_subset.csv contains both Fair-Speech (ethnicity column populated) and Common Voice (accent column populated) samples. Create separate manifests:

```python
import pandas as pd
eval_df = pd.read_csv("outputs/full-lora/eval_subset.csv")

# Fair-Speech samples have ethnicity column populated
fs_mask = eval_df["ethnicity"].notna() & (eval_df["ethnicity"] != "")
fs_eval = eval_df[fs_mask]
cv_eval = eval_df[~fs_mask]

fs_eval.to_csv("outputs/full-lora/eval_fairspeech.csv", index=False)
cv_eval.to_csv("outputs/full-lora/eval_commonvoice.csv", index=False)
print(f"Fair-Speech eval: {len(fs_eval)} samples")
print(f"Common Voice eval: {len(cv_eval)} samples")
```
