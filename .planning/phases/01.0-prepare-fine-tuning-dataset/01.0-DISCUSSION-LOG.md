## Train/Eval Split Strategy

| Option | Description | Selected |
|--------|-------------|----------|
| FS 80/20 stratified + CV test as second eval | Speaker-level stratified split of Fair-Speech by ethnicity. Ethnicity-aware GRPO training signal. CV test for accent eval. | ✓ |
| CV train for training + Fair-Speech full as eval | Large training set from CV official split, but no ethnicity labels during training. | |
| Hybrid two-stage | CV train for LoRA pretraining, then Fair-Speech split for GRPO. Most complex. | |

**User's choice:** FS 80/20 stratified + CV test as second eval (recommended option)
**Notes:** Fair-Speech is the only dataset with ethnicity labels, and the +203% ethnicity gap is the core research question. Speaker-level split prevents data leakage.

## Demographic Label Handling

| Option | Description | Selected |
|--------|-------------|----------|
| Parallel axes, no alignment | Keep native labels per dataset. FS: ethnicity/age/L1. CV: accent/gender/age. Reward function parameterized by axis. | ✓ |
| Unified superset schema | Single manifest with all columns, missing labels stored as empty, reward skips missing. | |
| Coarsen to shared axes only | Map to gender/age only. Loses ethnicity and accent — the primary targets. | |

**User's choice:** Parallel axes, no alignment (recommended option)
**Notes:** Ethnicity is Fair-Speech-only, accent is Common Voice-only. No cross-dataset alignment can unify these axes. Existing label maps (ETHNICITY_MAP, ACCENT_MAP) reusable as-is.

## Group Balance & Minimum Sizes

| Option | Description | Selected |
|--------|-------------|----------|
| Stratified oversampling to floor | Upsample small groups to ensure every batch has per-group signal. Monitor for memorization. | ✓ |
| Rolling-window accumulation | Keep natural distribution, smooth fairness reward over K batches. No data duplication. | |
| Drop groups below threshold | Exclude groups with <50 utterances from fairness term. Simplest but loses vulnerable groups. | |

**User's choice:** Stratified oversampling to floor (recommended option)
**Notes:** Ensures group presence in every batch for GRPO-08 stratified batching. Per-group train vs. eval WER divergence should be monitored.
