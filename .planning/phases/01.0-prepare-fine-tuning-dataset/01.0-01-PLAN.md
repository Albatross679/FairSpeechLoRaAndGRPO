4. **Mandatory checks** (per D-13):
   - **Check 1 -- Demographic label completeness**: For each axis in FS_DEMOGRAPHIC_AXES, count rows where value is NaN or empty string. Append to issues if >5% missing for ethnicity (primary axis).
   - **Check 2 -- MIN_GROUP_SIZE per split**: For each axis, compute `value_counts()` on both train and eval. Any group below MIN_GROUP_SIZE=50 in eval split is an issue. Below 50 in train is a warning.
   - **Check 3 -- Speaker/utterance leakage**: Load both CSVs, compute `set(train["speaker_id"]) & set(eval["speaker_id"])`. If overlap > 0, append CRITICAL issue with count. Also check `set(train["utterance_id"]) & set(eval["utterance_id"])` for utterance-level leakage.
   - **Check 4 -- Audio file existence**: Sample 100 random audio_path values from each split, check `os.path.isfile()`. If any missing, append issue with count.
   - **Check 5 -- Text normalization consistency**: Check that `sentence` column has no empty values where `sentence_raw` is non-empty (would indicate normalization failure). Check a sample of 50 sentences contain only lowercase ASCII + digits + punctuation (normalized text pattern).
   - **Check 6 -- CV manifest existence**: Verify cv_test_manifest.csv exists. If not, append warning.
   - **Check 7 -- Distribution drift**: For ethnicity axis, compare train vs eval proportions. If any group's proportion differs by more than 5 percentage points, append warning.

   print(f"\n{'='*60}")
