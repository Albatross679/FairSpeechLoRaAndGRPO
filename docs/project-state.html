Chapter 1 · current local processed data

1.1 Common Voice — complete local working set

Raw audio and manifest are local.
  • datasets/common_voice_24_clips/ — 16,398 MP3 clips.
  • datasets/common_voice_24_clips.zip — archive copy.
  • datasets/data/cv_test_manifest.csv — 16,398 rows with text, audio path, gender, accent, age, and client ID.
Clean and perturbed inference results are present.
  • 9 models.
  • 13 conditions per model: clean, 3 noise levels, 3 reverb levels, 3 silence levels, 3 masking levels.
  • Every Common Voice prediction CSV has 16,398 rows.

1.2 Common Voice results already computed

Artifact means a pipeline output we can inspect or reuse directly.
  • Prediction CSVs — one row per utterance: reference text, model transcript, WER, demographics, model name, and architecture label.
  • Metadata JSONs — run-level information: model ID, total utterances, overall WER, device, batch size, elapsed time, and timestamp.
  • Analysis outputs — computed summaries from predictions: group WERs, max/min fairness ratios, relative gaps, error decomposition, and plot/table files.
ArtifactWhat it containsStatus
datasets/results/commonvoice/Raw model outputs for CV: prediction CSVs and run metadata JSONs for all clean and perturbed runs.present
datasets/results/commonvoice/analysis/Clean-audio fairness analysis: summaries, WER-by-group plots, LaTeX tables, and error decomposition.present
datasets/results/tables/Machine-readable perturbation summaries: WER, insertion rate, and fairness-gap amplification by demographic axis.present
datasets/results/figures/Human-readable figures plus source CSVs for paper-style CV fairness and perturbation plots.present

1.5 FairSpeech — not locally complete

Current local state: FairSpeech underlying prediction CSVs and full analysis JSON are not present. Only downstream figure artifacts are visible, such as fig1_ethnicity_heatmap.

Caveats

  • CV perturbation results exist. We have prediction tables and analysis outputs, not necessarily stored perturbed audio files.
  • CV demographics are sparse. Accent/gender/age analyses use labeled subsets, not all 16,398 clips.
  • Resolve source-of-truth mismatch before final claims. summary.csv and summary_7models.csv disagree for Granite-8B.
  • FairSpeech needs teammate data or rerun access before making local ethnicity-result claims from raw tables.
